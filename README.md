API REST desarrollada en .NET para la gestión de estudiantes, profesores y calificaciones. 
Implementa arquitectura por capas, validaciones de negocio con excepciones controladas, paginación y filtros en servidor, documentación con Swagger 
y soporte de despliegue con Docker y Kubernetes.
